# To Do List App

This is a simple to-do-app using react.js


## Run Locally

Clone the project

```bash
  git clone https://github.com/tsepas/to-do-list
```

Go to the project directory

```bash
  cd to-do-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
